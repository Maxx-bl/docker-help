Introduction à Docker

Images · Conteneurs · Réseau · Dockerfile

▼
Nettoyage initial
docker ps -aLister tous les conteneurs
docker rm --force $(docker ps -aq)Supprimer tous les conteneurs
docker images -aLister toutes les images
docker rmi --force $(docker images -aq)Supprimer toutes les images
Images
docker pull alpineTélécharger une image
docker pull debian
docker pull nginx
docker load -i /chemin/image.tar.gzCharger depuis un fichier local
docker rmi nginx-helloSupprimer une image
Conteneurs — cycle de vie
docker run -ti --name alp1 alpineCréer + lancer (interactif)
docker run --name ng1 nginxCréer + lancer (détaché)
docker run --rm -ti alpineAuto-supprimé à l'arrêt
docker kill alp1Arrêter brutalement
docker stop alp1Arrêt propre
docker rm alp1Supprimer un conteneur
docker exec alp1 wc -l /usr/share/dict/frenchLancer une cmd dans un conteneur
Réseau · Partage de fichiers
docker inspect ng1Infos détaillées (IP, montages…)
docker run -p 8080:80 nginxPublier le port 80 → 8080
docker run -v ~/docker-share:/mnt alpineMonter un répertoire hôte
docker cp alp1:/anything ./anythingCopier fichier conteneur → hôte
